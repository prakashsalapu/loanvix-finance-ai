## LoanVix

LoanVix is a loan analysis tool that helps users understand and optimize their loan repayment journey. It provides detailed calculations, repayment insights, and prepayment analysis to support better financial decisions.

### Features

* EMI Calculator
* Amortization Schedule
* Loan Prepayment Analyzer
* EMI Increase Analysis
* Tenure Reduction Calculator
* Scenario Comparison
* Interactive Charts & Visualizations
* PDF Report Generation

### Tech Stack

* React.js
* Vite
* Tailwind CSS
* Framer Motion
* Recharts

### Purpose

 - LoanVix was built to help borrowers analyze loans beyond basic EMI calculations. Users can explore different repayment strategies, estimate interest savings, and understand the impact of prepayments and EMI changes.

### Installation

```bash
git clone <repository-url>
cd loanvix
npm install
npm run dev
```
